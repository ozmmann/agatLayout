Agat Inter Site
-------------------------

+ **Project Structure**
```
-dev
  |-fonts                       fonts directory
     |-roboto                   base materialize fonts
  |-html                        base html pages directory
     |-includes                 html parts directory(will be included in base html page)
  |-img                         decor images must be here
  |-js                          js directory
  |-sass                        base sass files directory
     |-components               materialize sass components
     |-my_style_components      directory for custom project style components
```

+ **Init Project**

```
1. Nodejs must be installed
2. Run command [npm install] in console (you must be in project directory)
3. Run command [gulp] in console (you must be in prgoject/node_modules/.bin)
4. Enjoy!
```
